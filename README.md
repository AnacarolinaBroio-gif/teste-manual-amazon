# 🧪 Teste Manual - Amazon

## 📌 Sobre o projeto
Este projeto tem como objetivo realizar testes manuais na plataforma Amazon, validando funcionalidades principais do sistema.

## 🔍 Funcionalidades testadas
- Home (página inicial)
- Barra de busca
- Navegação
- Carrinho de compras

## 🧪 Tipos de testes realizados
- Testes funcionais
- Testes exploratórios
- Testes de regressão

## 🐞 Resultados
- 44 casos de teste executados
- 43 passaram
- 1 falhou (bug identificado no parcelamento do carrinho)

## 🛠️ Ferramentas utilizadas
- Trello (organização)
- Excel (casos de teste e relatórios)
