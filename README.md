# 🧪 Projeto de Testes Manuais — E-commerce (Amazon)

Este projeto foi desenvolvido com o objetivo de aplicar na prática conhecimentos em Quality Assurance (QA), realizando testes manuais em uma aplicação real de e-commerce.

## 📌 Objetivo
Validar funcionalidades do sistema, garantindo qualidade, consistência das informações e uma boa experiência para o usuário.

## 🧪 Escopo dos testes
- Página inicial (Home)
- Barra de busca
- Menu de navegação
- Banners e carrossel
- Carrinho de compras

## 📊 Resultados dos testes
- 44 casos de teste executados
- 43 testes passaram
- 1 teste falhou

## 🐞 Bug identificado
Inconsistência no valor de parcelamento no carrinho, que não é atualizado ao alterar a quantidade de produtos.

## 🛠️ Atividades realizadas
- Criação de casos de teste
- Execução de testes manuais
- Documentação de bugs
- Elaboração de test report
- Registro de evidências (prints e vídeos)
